# CI/CD with GitHub Actions

This repository demonstrates a simple CI pipeline using GitHub Actions
to automatically test Python code on every push and pull request.

## Why this project
CI/CD pipelines are a core part of DevOps. This project shows how automated
testing can be integrated into a development workflow using GitHub Actions.

## Tech Stack
- Python
- GitHub Actions
- Pytest

## Pipeline overview
- Triggered on push and pull requests
- Installs dependencies
- Runs automated tests
